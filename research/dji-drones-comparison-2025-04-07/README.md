## 🎯 Особливості

### FPV vs Звичайний Польот
- **DJI Neo 2**: Підтримує FPV режим через окуляри (N3/Goggles 3), але орієнтований переважно на автоматичні режими і традиційний польот. Має новий ручний FPV режим через оновлення firmware (листопад 2025).
- **DJI Avata 2**: Чистий FPV дрон, створений для іммерсивного польоту. Повна підтримка ручного режиму (Manual/Acro), Motion Controller, FPV Remote Controller 3 для точного керування.

### Система Безпеки та Сенсори
- **DJI Neo 2**: **Омнінаправлена система виявлення перешкод** - монокулярний зір + LiDAR вперед + інфрачервоний датник вниз. Виявляє перешкоди з усіх боків.
- **DJI Avata 2**: **Обмежена система виявлення** - тільки внизу та назад. Немає фронтальних сенсорів для виявлення перешкод попереду.

### Автоматичні Режими
- **DJI Neo 2**: 
  - Palm Takeoff/Landing (зліт/посадка з долоні)
  - Gesture Control (керування жестами)
  - ActiveTrack 360° (відстеження об'єктів)
  - QuickShots (Dronie, Circle, Rocket, Helix, Boomerang)
  - SelfieShot
  
- **DJI Avata 2**:
  - Turtle Mode (переворот після падіння)
  - One-Push Acrobatics (автоматичні трюки)
  - Підтримка ручного режиму

### Захист Пропелерів
- **DJI Neo 2**: **Вбудований захист пропелерів** (включений у компактний дизайн), можна знімати для зменшення ваги.
- **DJI Avata 2**: **Вбудований захист**, не знімається, але забезпечує кращий захист під час ударів.

## 👥 Для Кого?

### DJI Neo 2 - Рекомендовано для:
- **Початківців** - найпростіший у керуванні, автоматичні режими, омнінаправлені сенсори
- **Контент-мейкерів** - компактний, легкий, QuickShots, ActiveTrack
- **Подорожей** - найлегший (151г), не потребує реєстрації (C0), компактний
- **Селфі та влогів** - Palm Takeoff, Gesture Control
- **Внутрішніх приміщень** - маленький розмір, захист пропелерів

### DJI Avata 2 - Рекомендовано для:
- **FPV пілотів** - справжній FPV досвід, ручний режим, Motion Control
- **Професіоналів** - краща камера, D-Log M, більша дальність
- **FPV гонок та трюків** - висока швидкість (27 м/с), Turtle Mode
- **Кіношників** - ширший кут огляду (155°), краща стабілізація, професійні кольори
- **Досвідчених пілотів** - хоче контролю та можливостей для творчості

## 🎯 Сценарії Використання

### Відео/Фото Контент
- **Neo 2**: Ідеально для влогів, селфі, подорожей. Автоматичні режими роблять зйомку простою.
- **Avata 2**: Краща якість зображення, D-Log M, більший динамічний діапазон - для професійного контенту.

### FPV Гонки та Трюки
- **Neo 2**: Підтримує FPV, але обмежена швидкість та відсутність ручного режиму спочатку (додано пізніше).
- **Avata 2**: **Очевидний вибір** - справжній FPV, висока швидкість, Manual режим, Turtle Mode.

### Внутрішні Приміщення
- **Neo 2**: **Кращий вибір** - компактний, легкий, омнінаправлені сенсори, Palm Takeoff.
- **Avata 2**: Можливий, але важчий та без фронтальних сенсорів - вищий ризик пошкоджень.

### Вулична Зйомка
- **Neo 2**: Хороший для простих сцен, але обмежена швидкість та дальність.
- **Avata 2**: **Кращий вибір** - більша дальність, вища швидкість, краща стабілізація.

### Вітростійкість
- Обидва: Рівень 5 (10.7 м/с) - однакова вітростійкість.
